Reshuffle App OAuth Redirects

LinkedIn Authentication

This site handles OAuth redirects for the Reshuffle app.

The LinkedIn redirect is configured at: /linkedin-callback/

The callback redirects to the app scheme: reshuffle://linkedin-callback

Technical Details

This is a static site that handles the OAuth 2.0 flow for social media integrations:

  1. User authenticates with the social platform
  2. Platform redirects to this site with authorization code
  3. This site extracts the code and redirects to the app using a custom URL scheme
  4. The app processes the code and completes authentication
